Links

Links are like little springboards your computer uses to hop from one page to another. Without those, the user of your website would be confined to just the title screen, and that would be sad! Links are added using the <a> anchor element. See the table below for an example:

Opening tag Attribute Content Closing tag
<a href="yourlink.com"> Go to yourlink.com to go nowhere! </a>
